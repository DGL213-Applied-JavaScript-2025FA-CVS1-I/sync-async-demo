How This Works:

1. Sequential Loading (Slow) - Waits for each request:

async function loadSequential() { const startTime = Date.now(); // Waits for user before fetching posts const user = await fetchUser(); // Waits for posts before fetching todos const posts = await fetchPosts(); // Waits for todos to finish const todos = await fetchTodos(); const totalTime = Date.now() - startTime; // Total: ~3 seconds (1s + 1s + 1s) }

2. Parallel Loading (Fast) - All requests at once:

async function loadParallel() { const startTime = Date.now(); // Start ALL requests simultaneously! const [user, posts, todos] = await Promise.all([ fetchUser(), // Starts immediately fetchPosts(), // Starts immediately fetchTodos() // Starts immediately ]); const totalTime = Date.now() - startTime; // Total: ~1 second (all run together!) }

Key Concepts:

  • Sequential (await one by one): Safe when operations depend on each other
  • Parallel (Promise.all): Much faster when operations are independent
  • Real API: Uses JSONPlaceholder (fake REST API for testing)
  • Error Handling: try-catch blocks prevent crashes
  • Loading States: Shows spinners while fetching data
